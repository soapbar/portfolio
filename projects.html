Created an 8-bit sigma-delta ADC, consisting of analog and digital components. With a full adder cell as a base component, made an FIR filter to remove ripples from the previous filter stage.

Neural recording array front-end

We made a neutral recording array schematic and layout in Cadence that takes the raw inputs of the brain and amplifies the signals with a low noise amplifier, then passes the signals through a bandpass filter to eliminated the unwanted noise, and finally passed the 16 channels to a 16:1 analog multiplexer to select a single signal to output.

Read the report here.
See the PPT here.


Go 2 Class 4 Me Robot

The goal for this project is to wirelessly be able to control a robot using a phone from any location in a building. The user will be able to see and hear what the robot sees and hears through the phone interface. In the end, this system could possibly attend class for a student, amongst other things! We used an Android phone's accelerometer to control the movement of the Raspberry Pi robot.

See the website here.

Digital Logic and Computer Organization

Learned about transistor network design, Boolean algebra, combinational circuits, sequential circuits, finite state machine design, processor pipelines, and memory hierarchy.
